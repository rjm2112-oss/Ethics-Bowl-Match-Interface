Hardcoded Ethics Bowl match rules
  • Judges score each participant out of 60 using the official sheet: presentation /15 (3 criteria, each 1–5), response to feedback /10, responses to judges /20, commentary /10, respectful dialogue /5.
  • Judges award votes based on the higher total score for each judge.
  • If both sides receive 1.5 votes, the match is a tie.
  • The app uses the full official online Ethics Bowl timings:
  • Presentation confer: 3:00
  • Presentation: 5:00
  • Commentary confer: 3:00
  • Commentary: 3:00
  • Response confer: 3:00
  • Response: 3:00
  • Judges' period total: 10:00
  • Each judge question: 1:00
  • Each judge answer: 2:20
